# 国内求职链接助手

一个本地静态网页工具，用来减少重复查找招聘网址的时间。页面分成两块：先按企业类型展示官网招聘入口，再按岗位、城市、公司搜索相关企业和招聘平台链接。

## 使用方式

直接双击打开 `index.html`，或在浏览器中打开这个文件。

## 功能

- 首批内置中国境内常见招聘企业
- 按企业类型分组展示官网招聘入口
- 支持企业名、别名、岗位关键词搜索
- 支持“企业 + 岗位”组合搜索
- 支持城市筛选
- 搜索结果优先展示可能相关的企业，并提供官网招聘、官网岗位搜索和招聘平台链接
- 支持收藏企业
- 支持记录投递状态
- 支持导出 CSV 投递表

## 当前边界

这个版本是纯本地静态网页，不会登录招聘网站，也不会实时爬取岗位详情。搜索结果里的“相关企业”来自本地企业库和岗位/行业匹配规则；真实开放岗位以企业官网和招聘平台打开后的页面为准。

如果后续要做实时更新，需要增加后端服务、定时任务、数据存储和合规的数据来源，优先考虑企业官网公开页面或手动 CSV 导入，不建议直接抓取需要登录或有反爬限制的平台页面。

## 文件说明

- `index.html`：页面结构
- `styles.css`：页面样式
- `app.js`：搜索、收藏、状态和导出逻辑
- `companies.js`：企业初始数据
